# Hybrid Healthy Lifestyle companion

An AI-powered healthcare assistant that predicts possible diseases based on user symptoms and provides personalized health recommendations. The project combines Machine Learning, data analytics, and a modern web interface to create an intelligent healthcare support system.

---

## Features

- Disease prediction using Machine Learning
- Personalized health recommendations
- User-friendly web interface
- Symptom-based prediction system
- Interactive frontend using React
- Backend API integration
- Health data processing and analytics
- Database integration for storing records
- Responsive UI for better user experience
